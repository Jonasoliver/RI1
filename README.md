# Vingadores vs Thanos

Projeto em JavaScript que cria objetos representando os Vingadores (Capitão América, Homem de Ferro, Hulk, Thor, Viúva Negra e Gavião Arqueiro) e o vilão Thanos. Todos os personagens são armazenados em um array e o código compara seus atributos (velocidade, força e resistência) usando estruturas de repetição, exibindo a descrição de cada um e indicando quem vence em cada categoria.

## Como executar

```powershell
node vingadores.js
```
